# BlockMatching
This is an implementation of  [A comparison of block-matching motion estimation algorithms](https://ieeexplore.ieee.org/document/6398002) paper.

## Dependencies
* OpenCV 4.2.0
* VTK 8.2.0
* GLEW 2.1.0
